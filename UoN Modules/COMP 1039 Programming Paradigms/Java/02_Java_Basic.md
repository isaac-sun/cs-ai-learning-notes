# Java002: Basics
## Program Control Statement
- Selection Statement
    - Choose different path to execute (e.g. `if`, `if-else`, `switch`)
- Iteration Statement
    - Repeat a section of code (e.g. `for`, `while`, `do-while`)
- Jump Statement
    - Transfer control to another place (e.g. `break`, `continue`, `return`)

More contents about control statement can be seen at module slides.

## Array
### What is an array?
An array is a collection of variables of the same type.
- It has a **fixed size**, cannot be changed after creation.
- Elements are accessed using an index (starting at 0)
- It can be both one-dimensional or multidimensional
- In java, arrays are objects
---

**General Form**
```
type[] arrayName = new type[size];
int[] numbers = new int[3];
```
